Navbar Organism

Complete navigation bar for Grenoble Roller community platform

Components
  • Left: Logo + Main menu (Accueil, Événements, Créer, Ressources)
  • Center: Search bar
  • Right: User menu (Profile dropdown) + Auth buttons
  • Responsive: Burger menu on mobile
UX Best Practices
  • Primary actions on the left (Accueil, Événements, Créer)
  • User-specific actions on the right (Profile, Logout)
  • Clear and simple labels
  • Icons for quick identification
  • Logical grouping (Resources in dropdown)

HTML Code

Copy this code to use the navbar in your application.

<nav class="navbar navbar-grenoble-roller navbar-expand-lg navbar-liquid shadow-liquid">
    <div class="container-fluid">
        <!-- Logo -->
        <a class="navbar-brand fw-bold d-flex align-items-center" href="#">
            <i class="bi bi-roller-skate me-2 fs-4"></i>
            <span>Grenoble Roller</span>
        </a>
        
        <!-- Toggle -->
        <button class="navbar-toggler" data-bs-toggle="collapse" 
                data-bs-target="#navbarGrenobleRoller">
            <span class="navbar-toggler-icon"></span>
        </button>
        
        <!-- Content -->
        <div class="collapse navbar-collapse" id="navbarGrenobleRoller">
            <!-- Main menu, Search, User menu... -->
        </div>
    </div>
</nav>
